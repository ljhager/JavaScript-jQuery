Basic JavaScript

JavaScript code for creating variables, performing operations, concatenation, indexed arrays, and adding elements to arrays.



More Basic JavaScript

JavaScript code for creating if, if-else, and else-if statements; while and do-while loops; and functions and arguments.



Open Links in New Tab

JavaScript code for creating links that when selected opens a new tab instead of replacing what is on the current page.


Internal Link: Hover the cursor over the link and a tool-tip will appear displaying the value of the HREF, as well as its URL at the bottom right of the screen.


External Link: Hover the cursor over the link and its web address display at the bottom right of the screen.



Color Theory Page Links

JavaScript and jQuery code hides all the paragraphs except the first one upon page load. When a link is clicked the background color changes to the color that was selected. The image is replaced by a sample of an object of that color and text is displayed that gives some information about that particular color.



Rotate Images

JavaScript and jQuery code rotates the display of four differently-sized images. Each image fades in then fades out; the size and shape of the DIV changes with each image. When the page loads it always begins with the same image.
